# GeneticNQueens
Genetic algorithim to solve the n-Queens problem.
To build: 
<code>make</code>
To run:
<code>./genQueens</code>

Defaulted to board size of 8 and a population size of 200.
Can change population size in main.cpp, line 5.
Can change board size in state.h, line 5.
To remake:
<code>make</code>
